Alt Drag.app
============

Linux-style alt+drag for windows on Mac OS X

Quick Guide
-----------

    git clone git://github.com/bahamas10/Alt-Drag.git
    cd Alt-Drag
    xcodebuild
    open build/Release/Alt\ Drag.app

Usage
-----

When the app is open use the hotkeys:
- ⇧ Shift + ⌘Command - Move
- ⇧ Shift + ⌥ Option - Resize
